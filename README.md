# Simple Redirect Page
A clean, customizable redirect page that can be easily deployed to Vercel. (Mostly AI coded)

## Features

- Simple, responsive design
- Configurable redirect URL and site name
- Countdown timer
- Works with JavaScript disabled (fallback to meta refresh)
- Easy deployment to Vercel

## How to use

1. [Clone to Vercel](https://vercel.com/new) by entering the git clone URL in the top
2. Deploy
3. Set environment variables in Vercel dashboard:
   - `REDIRECT_URL`: Your target URL
   - `SITE_NAME`: Display name for the site

   eg. REDIRECT_URL=https://google.com
   eg. SITE_NAME=Google
