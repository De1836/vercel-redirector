# Simple Redirect Page
A clean, customizable redirect page that can be easily deployed to Vercel. (Mostly AI coded)

## Features
- Easy to deploy
- Configurable
- You can put it on a qr code and change the redirect url and site name :)

## How to use
1. Clone to [Vercel](https://vercel.com/new) by entering repo URL
2. Deploy
3. Set environment variables in Vercel dashboard:
   - `REDIRECT_URL`: Your target URL
   - `SITE_NAME`: Display name for the site

   eg. REDIRECT_URL=https://google.com
       SITE_NAME=Google
